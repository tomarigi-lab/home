# CLAUDE.md

## このプロジェクト

`home` は、とまり木の正式ホームページとGitHub Pages公開用のプロジェクト。

## 作業開始時に必ず読む

- 対象の依頼書: `handoff/requests/`
- 現在地: `handoff/project-status.md`
- 詳細ルール: `project/project-guidelines.md`

## 基本ルール

- このプロジェクト内だけを読み書きする。
- 依頼書の範囲と禁止事項を守る。
- 未決事項を勝手に確定しない。
- 作業結果は `handoff/results/` に残す。
- 作業後は `handoff/project-status.md` を最新状態へ更新する。
- Webサイトとして公開するファイルは `docs/` 内に置く。
- Git push、公開、削除、履歴書き換え、外部送信は、依頼書に明示的な許可がある場合だけ行う。
